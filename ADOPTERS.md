# SkillBag Adopters

This page tracks public adoption, implementation work, and ecosystem support
for SkillBag.

## Listed Projects

Projects may be listed here when they publicly use, implement, or support
SkillBag.

To add a project, open a pull request with:

- project or organization name
- public URL
- adoption type
- short description of how SkillBag is used
- contact or maintainer handle, if public

## Adoption Types

Use one of these categories:

- `workspace`: repository uses `SKILLBAG.md` and `.skills/`
- `source`: repository publishes installable SkillBag skills
- `implementation`: agent, validator, installer, or tool supports SkillBag
- `supporter`: organization publicly supports the standard
- `sponsor`: organization funds shared project work

## Support Statements

No independent vendor support statements are listed yet.

Support statements should be public and specific. A useful statement explains
whether the organization is evaluating, implementing, sponsoring, or adopting
SkillBag.

## Removal

Entries may be removed when:

- the linked project no longer exists
- the project no longer uses or supports SkillBag
- the entry is misleading or promotional without technical relevance
- the organization asks to be removed
