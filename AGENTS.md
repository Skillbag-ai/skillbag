SkillBag Specification

This repository contains the normative SkillBag specification.

Read and execute `SKILLBAG.md` before any other work.

- Treat `SKILLBAG.md` as the authoritative normative source.
- Prefer precise definitions over broad descriptions.
- Make precedence, override, and installation behavior explicit.
- Identify ambiguities and edge cases rather than glossing over them.
- Preserve compatibility with the base skill format where practical.
- Use examples only when they remove ambiguity.
- Keep all documentation consistent with `SKILLBAG.md`.
